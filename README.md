# 🎮 Tic Tac Toe Game (HTML, CSS & JavaScript)

An interactive **Tic Tac Toe** web game built using **HTML**, **CSS**, and **JavaScript**.
This project demonstrates frontend development concepts such as DOM manipulation, event handling, and game logic implementation.

---

## 📌 Project Overview

This is a browser-based Tic Tac Toe game where two players take turns placing **X** and **O** on a 3×3 grid. The game automatically checks for winning combinations and detects draw situations.

The project was developed to practice:

* Web development fundamentals
* JavaScript game logic
* UI styling with CSS
* Interactive user experience

---

## ✨ Features

* ✅ Two-player gameplay
* ✅ Real-time win detection
* ✅ Draw detection
* ✅ Reset/restart functionality
* ✅ Clean and responsive interface
* ✅ Browser-based (no installation required)

## 🎯 Game Rules

1. The game is played on a 3×3 grid.
2. Player **O** starts first.
3. Players alternate turns placing symbols.
4. First player to align three marks wins.
5. If all cells fill without a winner, the game ends in a draw.
